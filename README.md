# Веб-интерфейс для оценки степени износа по фотографии

Это готовое приложение для запуска **без повторного обучения**. Оно использует уже сохранённые артефакты модели из notebook и поднимает web UI, в котором можно:

- загрузить фотографию дефекта;
- получить карту повреждений и бинарную маску;
- рассчитать степень износа по фото;
- при наличии hybrid-моделей дополнительно получить риск деградации и RUL-proxy.

## 1. Что нужно для запуска

Минимально нужен файл:

- `best_damage_segmentation_binary.pt`

Желательно также добавить:

- `binary_threshold_sweep.csv`
- `hybrid_wear_regressor.joblib`
- `hybrid_poor_within_horizon_clf.joblib`
- `hybrid_rul_regressor.joblib`
- `hybrid_feature_defaults.json`

## 2. Где взять артефакты

Они хранятся в папке `artifacts`.

## 3. Локальный запуск

```bash
pip install -r requirements.txt
python app.py
```

После запуска откроется локальный сайт, обычно по адресу:

```bash
http://127.0.0.1:7860
```

## 4. Что делает приложение

- Загружает обученную сегментационную модель.
- Строит бинарную маску повреждения.
- Вычисляет визуальные признаки: площадь, периметр, ширину, длину, фрагментацию.
- Рассчитывает визуальную степень износа по фото.
- При наличии hybrid-моделей рассчитывает дополнительные показатели по параметрам объекта.
